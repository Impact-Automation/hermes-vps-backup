# Nano Banana Pro

Image generation skill powered by Google Gemini 3 Pro Image.

## Tools

### `generate_image`
Generates a 16:9 cinematic image based on a text prompt.

**Parameters:**
- `prompt`: (string, required) Detailed description of the image to generate.

## Usage
"Imagine a futuristic city in the clouds"
"Generate an image of a cybernetic lobster"

## Requirements
- `GEMINI_API_KEY` set in environment.
- `node` installed.
